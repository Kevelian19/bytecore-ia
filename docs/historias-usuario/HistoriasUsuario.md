# HU-002 - Registro de Empresa

## Prioridad

Alta

## Actor

Emprendedor

## Descripción

Como emprendedor quiero registrar mi empresa para comenzar a utilizar ByteCore IA.

## Criterios de aceptación

- Debe solicitar el nombre de la empresa.
- Debe solicitar el NIT.
- Debe solicitar correo electrónico.
- Debe solicitar teléfono.
- Debe crear automáticamente el usuario administrador.
- El correo no puede estar registrado previamente.

## Reglas de negocio

- El correo debe ser único.
- El NIT no puede repetirse.

## Dependencias

Ninguna.

5 Story Points
