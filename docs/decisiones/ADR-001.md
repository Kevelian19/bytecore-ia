# ADR-001

## Título

Las ventas se generan a partir de pedidos confirmados.

## Estado

Aceptado

## Contexto

Las microempresas suelen confirmar manualmente las ventas antes de registrarlas.

## Decisión

El sistema creará inicialmente un pedido.

Solo cuando el administrador confirme el pedido se registrará la venta.

## Consecuencias

- Mayor control.
- Evita ventas falsas.
- Facilita futuras integraciones con pagos.
